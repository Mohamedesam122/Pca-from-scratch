# Pca-from-scratch
# Machine Learning Project – Naive Bayes with Feature Selection and PCA

## Overview
This project implements a Naive Bayes classifier from scratch using Python.  
The goal is to compare the performance of the model using three different approaches:

1. Baseline Naive Bayes
2. Naive Bayes with Feature Selection
3. Naive Bayes with PCA (Principal Component Analysis)

The project evaluates how dimensionality reduction and feature selection affect model accuracy.

---

## Dataset
Two datasets were used in this project:

- **Categorical Dataset**
- **Numerical Dataset**

The datasets are preprocessed before training the model.

---

## Features
- Implementation of **Naive Bayes from scratch**
- **Feature Selection** using statistical methods
- **PCA implementation from scratch**
- Accuracy comparison between different approaches
- Visualization using **Matplotlib**
